# Последнее

Ссылки:
- [`Updates`](/doc/ru/object/updates.md)

`Updates::last` означает, что есть только поле `last`.
Остальные отсутствуют.

```json
{
    "is_ok": true,
    "data": {
        "updates": Updates::last
    }
}
```
